Правила Сервера

1.0 ОБЩЕЕ

1.1. УважениеУважайте других игроков. Оскорбления, травля, расизм, дискриминация — бан.

1.2. ЧатНе спамьте и не флудите в чате. Реклама других серверов/проектов запрещена.

1.3. Личная информацияНе выдавайте личную информацию других людей (доксинг).

1.4. ОбщениеИспользуйте общий чат по назначению. Для личных разговоров есть /msg.

1.5. Ники и СкиныЗапрещены ники и скины, содержащие оскорбления, нецензурщину или провокационные темы.

1.6. БагиНашли баг? Сообщите админу. Использовать баги для выгоды нельзя.

1.7. НаказанияАдминистрация может выдавать наказание по своему усмотрению.

2.0 АДМИНИСТРАЦИЯ

2.1. Роль админовАдмины помогают, поддерживают порядок и решают технические проблемы. Они не ваши слуги.

2.2. РешенияРешение старшего админа — окончательное.

2.3. ЧестностьАдминам запрещено использовать свои возможности для личной выгоды в игре.

2.4. СвязьВопросы и жалобы админам — через тикеты в Discord, а не в игровой чат.

3.0 ИГРОКИ

3.1. ГриферствоЗапрещено: ломать/портить чужие постройки, убивать животных, воровать из сундуков, заливать лавой/водой.

3.2. ЧитыЗапрещены: X-Ray, Fly, Kill Aura, автокликеры, дюпы и любые программы, дающие преимущество. Бан навсегда.

3.3. PvPРазрешён только по взаимному согласию. Убивать игроков без их согласия нельзя.

3.4. ПриватностьНе стройтесь вплотную к чужим базам без спроса. Уважайте чужое пространство.

3.5. ЛагиФермы и механизмы не должны вызывать сильных лагов. Если сервер "лагает" из-за вашей постройки — вас попросят её отключить.

3.6. СпавнНе стройте постоянные базы и не мусорьте на спавне. Спавн — общественная зона.

4.0 Сервер

4.1. Разрешенные модыОптимизация (OptiFine, Sodium), графика (шейдеры), удобство (Minimap БЕЗ подсветки, Inventory Tweaks, Replay Mod).

4.2. Запрещенные модыЛюбые моды и программы, дающие игровое преимущество (X-Ray, автокликеры, полёт и т.д.).

4.3. ПлагиныНа сервере могут быть служебные плагины только для защиты (логирование) и приватов. Они не меняют ванильный геймплей.

4.4. Спорные ситуацииАдминистрация оставляет за собой право окончательного решения в спорных ситуациях, исходя из здравого смысла и интересов сообщества.

Часто задаваемые вопросы

Нужна ли лицензия?

Нет, на нашем сервере можно играть как с лицензией, так и с пиратки. Но мы рекомендуем использовать официальный лаунчер!

Как получить приват?

Приваты выдаются автоматически при установке блока привата. Подробнее читайте в разделе "Как попасть".

Есть ли вайпы?

Мы стараемся не делать вайпы без крайней необходимости. Мир живет годами, и ваши постройки останутся с вами.

Какие версии поддерживаются?

Мы поддерживаем последнюю стабильную верисю Minecraft Java Edition. Следите за новостями в Discord.
